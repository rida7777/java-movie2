# 우아한테크코스 코딩테스트 프로젝트 - 영화예매

## 기능 구현 목록
* [예약] 
	1. 영화목록을 출력한다.
	1. 영화번호를 입력한다.
		* [예외처리] 없는 영화번호를 입력했을 때
	1. 해당영화의 스케줄을 출력한다.
	1. 스케줄번호를 입력한다.
		* [예외처리] 1시간 이내의 영화인지 확인
		* [예외처리] 지나간 스케줄인지 확인
		* [예외처리] 없는 스케줄 인덱스를 입력했을 때
	1. 인원을 입력한다.
		* [예외처리] 수용인원을 초과할때
		* [예외처리] 입력인원이 0명 이하일 때
	1. 예약정보를 출력한다.
* [결제]
	1. 포인트를 입력한다.
	1. 결제수단을 입력한다.
	1. 결제금액을 출력한다.
		* 할인율 : 카드 5%, 현금 2%
